# 🔴 CrimeWatch — Urban Crime Intelligence System

> A forensic-themed Streamlit platform covering all 4 course units through a real investigative use case.

## 🎯 Syllabus Coverage

| Unit | Topic | Module |
|------|-------|--------|
| **1** | Python for Data Visualization | 3D Analytics + Crime Atlas |
| **2** | Python for Web Application | Every page (6 pages, all widgets) |
| **3** | Python for Image Processing | Evidence Lab (30+ forensic ops) |
| **4** | Python for Text Analysis | Report Intelligence (7 tabs) |

## 🚀 Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📄 The 6 Pages

1. **🎯 Command Center** — Platform overview, 6 KPI cards, top crime cities, crime breakdown
2. **🗺️ Crime Atlas** — 4 map views: hotspot bubbles, heatmap, clusters, state choropleth
3. **📊 3D Analytics** — 5 3D visualizations: bars, histograms, scatter, surface, animated race
4. **🖼️ Evidence Lab** — 30+ forensic image ops across 6 categories (with 2 demo CCTV images)
5. **📝 Report Intelligence** — 7 NLP tabs: normalize, classify (3 models), find similar cases, search, sentiment, n-grams, word cloud
6. **ℹ️ About** — Info on CrimeWatch & Tech stack

## 🎨 Theme

Forensic noir — crimson (#DC2626) on deep black (#0A0A0F), Playfair Display serif headers, Oswald for sections, JetBrains Mono for technical details. Evokes a police case-file aesthetic without being cartoonish.

## 🎤 Viva Flow (8 minutes)

1. Open sidebar → show 4 syllabus bullets
2. Command Center → 6 KPIs + top crime chart
3. Crime Atlas → Hotspot map (click Mumbai), then Heatmap
4. 3D Analytics → 3D Bars (change year), then Animated Race ▶
5. Evidence Lab → "Night Shot" demo → CLAHE → Canny → Face Detection
6. Report Intelligence → Classify a new crime report → show confusion matrix → Similar cases
7. About → show syllabus matrix

## 📊 Dataset

- **20 major Indian cities** with lat/long coordinates
- **10 crime categories**: Theft, Assault, Fraud, Burglary, Vandalism, Robbery, Cybercrime, Drug Offense, Homicide, Harassment
- **6 years** (2019–2024) × 12 months
- **8 measures per record**: Cases, Severity, Resolution Rate, Arrests, Response Time, Population
- **Synthetic** — generated by `generate_data.py`. Not real-world data.
